# Symbolic Regression on Noisy 1D Data (Cross-Validation)

## Problem Statement

Given 200 noisy (x, y) data points sampled from an unknown 1D function with Gaussian noise (sigma = 0.05) and x drawn uniformly from [-5, 5], recover a function f(x) that best fits the underlying signal. Each orbit is solved by 2 independent agents (parallel_agents: 2) for cross-validation.

## Data

- **Training set:** 200 points (x, y) with x ~ U[-5, 5], y = f_true(x) + N(0, 0.05^2)
- **Held-out test set:** 500 clean points (no noise), sampled from the same distribution and target function
- The training data lives at `research/eval/train.npz` (generated by the evaluator setup) and is visible to agents. The held-out test set lives at `research/eval/test.npz` and is used only by the evaluator.

## Solution Interface

Solution must be a Python file `solution.py` exposing a function `f(x)` that takes a 1-D numpy array and returns a 1-D numpy array of the same shape.

```python
# orbits/<name>/solution.py
import numpy as np
def f(x: np.ndarray) -> np.ndarray:
    ...
```

The evaluator will also read the training data from `research/eval/train.npz` when running.

## Success Metric

**MSE on held-out test set** (minimize). Target: MSE < 0.01. The Bayes-optimal floor is ≈ 0.0025 (noise variance on the training side); the held-out set is clean so MSE measures true function recovery.

Evaluator contract (already present under `research/eval/evaluator.py`):
```bash
python3 research/eval/evaluator.py --solution <path/to/solution.py> --seed <int>
# stdout: METRIC=<float>
```
